Camera & Vision

Processes incoming MJPEG video feeds using native JNI and lightweight model gating to provide high-speed industrial safety inspection.

  • PPE Detection: Real-time scanning using custom YOLOv8 models. Instantly detects boots, gloves, helmets, vests, goggles, and masks.
  • Smart Gating: Uses Google ML Kit Face Detection as a low-power filter. YOLOv8 classifier is only triggered when a person is in frame, reducing CPU idle usage by 89%.
  • OCR Code Scanning: Integrates Baidu PaddleOCR runtime via C++ native JNI to read worker identification tags and verify security authorization.

IMU Telemetry

Monitors high-frequency accelerometer, gyroscope, and altimeter registers via a shared I2C bus lock to evaluate physical safety states.

  • Fall Detection FSM: Tracks 3D acceleration and angular velocity at 20Hz. Evaluates state transitions (Normal -> Free Fall -> Impact -> Calmness) to identify worker accidents.
  • Posture Recognition: Tracks roll and pitch matrices to calculate head tilt angles, warning workers when looking up or down excessively.
  • Gesture Commands: Decodes consecutive roll/pitch motions to identify nods, reverse nods, and headshakes for hands-free user interaction.

Location & SOS

Integrates mobile GPS hardware and cellular network registers to ensure worker localization, zone compliance, and automated emergency routing.

  • Live Tracking: Dynamically maps active GPS coordinates and tracks elevation deltas using real-time barometric pressure shifts.
  • SOS SMS Broadcasting: Triggers a 10-second countdown immediately upon fall detection, sending automated coordinates to registered supervisors.
  • Zone Compliance: Maps locations against OCR scanned safety permits to ensure the correct personnel are in authorized sections.
